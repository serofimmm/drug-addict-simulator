## скачивание

### на linux дистрибутивы
```bash
git clone https://github.com/serofimmm/drug-addict-simulator.git
cd drug-addict-simulator
```

### Открытие

1. Запустите **Unity Hub**.
2. Нажмите **Add** → выберите папку проекта.
3. Откройте проект через Unity **6000.3.17f1**.
